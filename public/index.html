Convo Buddy is a free tool that helps ESL teachers get the conversation started.

This app features more than 300 hand-picked questions intended to help English language learners practice speaking. Questions can be generated one at a time or viewed all at once.

The main page is where the individual questions appear. Advance or go back a question by clicking the arrow buttons or swiping left or right. The counter (bottom center) indicates the total number of questions in the current search, and the unshuffled link below that will randomize the order in which they're shown.

To modify which questions are generated, click the categories link (top right). To see the full list of questions and which questions pertain to which categories, click the questions link (top right). Return to the main page by clicking the convo buddy link (top center).

If you would like to contribute any questions or give feedback, you can do so via the submit a question link (bottom left) or by emailing jdcoppola@gmail.com.
